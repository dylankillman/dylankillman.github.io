Professional Self-Assessment

Completing the Computer Science program at Southern New Hampshire University, along with developing this ePortfolio, has been quite a journey for me. Throughout the program, I’ve had the opportunity to grow not only my technical skills, but also my confidence and focus. The coursework and hands-on projects that I worked on during this program have helped me refine my abilities and define my professional goals more clearly. Revisiting projects I completed earlier in the program and enhancing them with the knowledge and skills I gained later on in the program gave me a valuable opportunity to demonstrate my understanding of core computer science principles, including software and UI design, data structures, and databases.

Throughout the program, I’ve learned the value of collaboration in development environments, and I’ve been able to apply that every day at work as well. There were a couple of team-based assignments throughout the program, and these allowed me to learn how to contribute meaningfully within a group, communicate my ideas clearly, and support others via peer review and positive or negative feedback. I also often had peers review my code and projects, even when they weren’t developed in a group environment, in order to further foster my collaborative skills. These experiences prepared me to work in teams and communicate effectively with both technical partners, along with non-technical stakeholders like a customer for whom I’m developing a product. I’ve also honed my skills in creating documentation for my products, along with clear, maintainable code with clear comments to improve readability and allow for other professionals to easily update and modify my code.

My work with data structures and algorithms in this program deeply improved my problem-solving skills and allowed me to create more efficient solutions to problems. For example, in my data structures enhancement, I redesigned my CS-300 Course Planner product to utilize a binary search tree instead of a vector for data storage and traversal. This not only improved the execution time but also demonstrated my ability to analyze trade-offs when it comes to design decisions—balancing performance, efficiency, and maintainability. This is vital when it comes to building scalable products.

The software engineering and database-focused courses helped solidify my understanding of the software development process from start to finish. I learned how to apply object-oriented principles to the software development lifecycle and combine that with the use of tools and frameworks to create efficient and modular products. In one of the enhancements for this capstone, I completely refactored a mobile application to align with more modern UI design practices and improve the usability and visual appeal. In another artifact, I improved a MongoDB-based dashboard product by adding indexing, aggregation pipelines, and bulk insertion techniques, which improved both usability and overall performance.

Security was also a big part of this program. Because of this, I’ve been able to develop a mindset that anticipates vulnerabilities and incorporates proactive measures instead of standing back and reacting when a security breach or incident might happen. I’ve learned about and applied secure coding practices from the requirements phase all the way through development and rollout of the product. These practices include validating user input, applying the principle of least privilege, and understanding the risks of generally non-secure practices like unsafe data handling. Even though security wasn’t the central focus of all artifacts, I still approach every project paying special attention to security.

Altogether, I feel that the enhancements that are included in this portfolio represent a full picture of my skills and growth as a developer throughout this program. Each artifact highlights different strengths—software engineering through user-centric designs, data optimization through more efficient data structures, and database fluency through performance tuning. These not only demonstrate my technical proficiency but also my ability to communicate and present my work in a coherent, professional manner.
